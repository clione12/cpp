# cpp03_notebook

流程控制

- if
    - 三目运算符 `: ?`
    - 可以被转换为bool的都可以作为condition，但是要考虑代码可读性
- while
    - while
    - do while
    - break
    - continue
    - 注意：=和==，别把condition写成死循环
    - size_t 是无符号整数，0-1是最大值
- for
    - 太常见了，没什么坑
- goto
    - 不要使用
- switch
    - 不要忘记break
    - 可以理解为：goto到某个case下面，这样就不会忘记break了
